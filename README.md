# ⛪ St. Joseph Church Management Web Application

A centralized and dynamic Church Management Web Application developed to digitally manage announcements, events, parish information, and media content.  

This system provides a secure admin panel for authorized administrators to manage church data while parish members can access real-time updates through a responsive web interface.

---

## 🚀 Features

### 🌐 Public Website
- View Announcements
- View Upcoming Events & Festivals
- Parish Council Member Details (Head, Subhead, Members)
- Old Priests Section
- Heart Convent Section
- Anbiyam Coordination Section
- Image Gallery
- Video Gallery
- Fully Responsive Design (Mobile & Desktop)

### 🔐 Admin Panel
- Secure Admin Authentication
- Add / Edit / Delete:
  - Announcements
  - Events
  - Festival Details
  - Parish Council Members
  - Old Priests
  - Images
  - Videos
- Dynamic Content Management
- Cloud-Based Media Handling
